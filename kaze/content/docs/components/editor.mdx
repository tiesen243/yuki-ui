## Usage

1. Create state to hold the Markdown content:

```tsx
const [markdown, setMarkdown] = useState('')
```

2. Render the `Editor` component and pass the Markdown content and change handler:

```tsx
<Editor value={markdown} onChange={setMarkdown}>
  <EditorInput />
  <EditorPreview />
</Editor>
```

3. Parse the existing Markdown content to HTML:

```tsx
const html = parseMarkdown(markdown)

<div dangerouslySetInnerHTML={{ __html: html }} />
```
